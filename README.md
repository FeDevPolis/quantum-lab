# Quantum Lab ⚛️

Este repositorio contiene mis primeros experimentos en **Computación Cuántica** utilizando Python y el framework **Qiskit** de IBM.

## 🚀 Primer Experimento: Hello Quantum
En el script `hello_quantum.py`, implementé un generador de números aleatorios cuánticos utilizando la propiedad de **Superposición**. 

### Conceptos aplicados:
* **Qubits:** La unidad mínima de información cuántica.
* **Compuerta Hadamard (H):** Coloca al qubit en un estado de superposición (50% de probabilidad para |0⟩ y |1⟩).
* **Medición:** El colapso del estado cuántico a un bit clásico.

## 🛠️ Instalación
1. Clonar el repositorio.
2. Crear un entorno virtual: `python -m venv venv`.
3. Instalar dependencias: `pip install -r requirements.txt`.
